Airport Runway Simulator

A Python-based aviation scheduling and simulation project

Project Overview

The Airport Runway Simulator is a Python-based system that simulates aircraft landing and take-off operations on a single or multiple runways. It models real-time airport behavior using scheduling algorithms, queue management, event handling, and safety separation rules. The simulation helps analyze delays, queue length, runway utilization, and emergency handling efficiency.

Key Features

  • Simulates landing and take-off sequences
  • Supports FCFS and Priority scheduling algorithms
  • Emergency aircraft handling
  • Real-time simulation with event logging
  • Visualization of runway usage and queue movement
  • Performance metrics such as waiting time and utilization

Architecture Summary

The system architecture includes modular components such as Aircraft Generator, Event Queue, Scheduler Module, Runway Manager, Simulation Clock, and Visualization Layer. Aircraft requests enter the system through the generator, are scheduled by the scheduler, and processed through the runway manager before results are displayed visually.
